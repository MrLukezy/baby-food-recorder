# 宝宝饮食记录 (Baby Food Recorder)

一款帮助家长记录宝宝每日饮食的应用，未来支持微信小程序。

## 技术栈

- **Vite** - 极速构建工具
- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 原子化 CSS

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
