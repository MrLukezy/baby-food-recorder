## 🎯 核心功能更新

### 1. 三天排敏法智能状态判断

#### 排敏规则
根据权威指南，每种新食物应连续观察 **3 天**：

| 状态 | 条件 | 显示标签 |
|------|------|----------|
| **排敏完成** | 连续 3 天无不良反应 | `已排敏 ✓`（绿色） |
| **排敏中** | 不足 3 天且无过敏反应 | `排敏中 X/3天`（黄色） |
| **过敏** | 出现任何过敏反应 | `过敏 ✕`（红色） |

#### 技术实现
- **新增函数** `getFoodAllergenStatus(foodId)`：自动统计某食物的排敏天数
- **更新统计逻辑** `getStats()`：按食物去重统计排敏状态
- **进度可视化**：排敏中的食物显示 3 个圆点进度条

#### 用户价值
- 无需手动判断排敏是否完成，App 自动识别
- 清晰展示每种食物的排敏进度
- 避免误将"排敏中"的食物当作"安全食物"

### 2. 返回键导航支持

#### 问题背景
用户在使用浏览器返回键时，App 无法正确回退到上一页，只能退出整个应用。

#### 解决方案
- 集成浏览器 History API（`pushState` / `popstate`）
- 每次页面切换（Tab 切换、进入分类详情）都会记录历史状态
- 监听浏览器返回键，正确回退到上一个页面

#### 技术细节
```typescript
// 页面变化时推入历史状态
useEffect(() => {
  if (page.type === 'tab') {
    window.history.pushState({ tab: page.tab }, '');
  } else if (page.type === 'category_detail') {
    window.history.pushState({ categoryId: page.categoryId }, '');
  }
}, [page]);

// 监听返回键
useEffect(() => {
  const handler = (e: PopStateEvent) => {
    if (e.state) setPage(e.state);
  };
  window.addEventListener('popstate', handler);
  return () => window.removeEventListener('popstate', handler);
}, []);
```

#### 支持场景
- ✅ 首页 → 日历 → 返回到首页
- ✅ 首页 → 食物分类详情 → 返回到首页
- ✅ 任意 Tab → 详情页 → 返回到对应 Tab
- ❌ 引导页（创建宝宝/选择食物）不集成历史导航（一次性流程）

### 4. 同时排敏多食物警告

#### 医学背景
根据 AAP 指南，**每次只能引入一种新食物**，否则无法判断过敏源。

#### 警告逻辑
- 当用户尝试添加新食物时，检查是否有其他食物正在排敏中（不足 3 天）
- 如果存在，弹出确认对话框提醒用户
- 用户可选择继续添加（不强制阻止）

#### 警告弹窗内容
```
💡 排敏提醒

当前已有食物正在排敏中：
南瓜、胡萝卜

不建议同时排敏两种食物，这样无法判断过敏源。
建议等当前食物排敏完成（连续3天）后再引入新食物。

确定要继续添加吗？
```

#### 技术实现
```typescript
const observingFoods = getObservingFoods();
if (observingFoods.length > 0 && !isAlreadyObservingThis) {
  const confirmed = window.confirm('...');
  if (!confirmed) return;
}
```

#### 新增函数
- `getObservingFoods()`：返回所有正在排敏中的食物列表

## 🔬 婴儿排敏指导方案调研

### 调研来源
- **AAP（美国儿科学会）**：Starting Solid Foods (2026.06 更新)
- **NHS（英国国民保健署）**：Best Start in Life - Weaning Guidelines
- **中国卫健委**：《7-24月龄婴幼儿喂养指南》
- **LEAP Study**：Learning Early About Peanut Allergy (NEJM 2015)

### 核心结论

#### 1. 排敏标准流程
```
Day 1: 少量（1/4~半勺）→ 观察2-4小时 → 记录
Day 2: 加量（1-2勺）  → 观察2-4小时 → 记录
Day 3: 正常量        → 观察2-4小时 → 确认结果
```

#### 2. 常见过敏症状

| 系统 | 症状 | 严重程度 |
|------|------|---------|
| 皮肤 | 口周/面部红疹、荨麻疹、湿疹加重、皮肤瘙痒 | 轻~中 |
| 消化 | 呕吐、腹泻、便血、腹胀、拒食、水样便 | 中 |
| 呼吸 | 打喷嚏、流涕、咳嗽、喘息、呼吸困难 | **重（就医）** |
| 全身 | 嗜睡、烦躁、发热、面色苍白、休克 | **危急（急救）** |

#### 3. 排敏结果判定
- **排敏成功（安全）**：连续 3 天无任何上述症状 → 可纳入常规食谱
- **需继续观察**：出现轻微皮疹或轻微消化不适 → 停用 1-2 周后再试
- **排敏失败（过敏）**：出现中度以上反应 → 立即停用，记录症状，3个月后可再尝试

#### 4. 关键注意事项
- ✅ **每次只引入 1 种新食物**（绝对不要同时引入 2 种以上）
- ✅ **新食物安排在白天**（上午或中午，避免晚间）
- ✅ **生病/接种疫苗期间不引入新食物**
- ✅ **高敏食物需单独排敏**（鸡蛋蛋白、虾蟹、花生坚果、芒果、猕猴桃、小麦）

#### 5. 高敏食物最新共识（AAP 2019+）
> 没有证据表明推迟引入高敏食物（如鸡蛋、花生、鱼、牛奶制品）超过 4-6 个月可以预防过敏。反而有研究表明**早期引入花生制品可能降低花生过敏风险**（LEAP 研究）。但严重湿疹或鸡蛋过敏的婴儿建议先做花生过敏测试。

#### 6. 不同月龄段辅食添加建议

| 月龄 | 食物性状 | 推荐食物 | 排敏节奏 |
|------|---------|---------|---------|
| **6 月** | 泥糊状（细腻） | 米粉、南瓜、胡萝卜、土豆、苹果、香蕉、牛油果 | 每 3 天 1 种新食物，从低敏蔬菜/水果开始 |
| **7-8 月** | 细碎/软烂 | 鸡肉泥、蛋黄、西蓝花、菠菜、小米粥、酸奶 | 开始尝试肉蛋类，蛋黄先于蛋白 |
| **9-10 月** | 小块/软颗粒 | 牛肉、鱼肉、面条、豆腐、桃子、手指食物 | 可加快引入节奏（2-3天），开始尝试中敏食物 |
| **11-12 月** | 接近成人/切小块 | 虾仁、蛋白、全麦面包、奶酪、更多水果 | 可尝试高敏食物，但须单独排敏 |
| **12+ 月** | 成人饮食 adapted | 牛奶、坚果碎、贝壳类、鱿鱼等 | 基本完成基础排敏，可尝试所有食物 |
